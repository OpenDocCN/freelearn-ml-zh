# 第三部分：R 中的数学统计学基础

随着我们开始本书的第三部分，我们将从*第二部分*中探讨的数学基础过渡到高级统计方法和模型构建的领域。虽然前面的部分在基本统计和数学基础方面奠定了基础，但本部分旨在将你的能力提升到更高级的水平，同时利用 R 的强大功能进行实际应用。

到本部分结束时，你将掌握一套强大的高级统计和建模技能。本部分提供了你进入数据科学更高级主题所需的先进知识和实践经验。

本部分包含以下章节：

+   *第十章*, *概率基础*

+   *第十一章*, *统计估计*

+   *第十二章*, *R 中的线性回归*

+   *第十三章*, *R 中的逻辑回归*

+   *第十四章*, *贝叶斯统计*
