# 前言

*使用 Python 学习数据挖掘*的第二次修订是针对程序员编写的。它的目标是向广泛的程序员介绍数据挖掘，因为我认为这对于计算机科学领域的所有人来说都是至关重要的。数据挖掘正在迅速成为下一代人工智能系统的基石。即使你发现自己没有构建这些系统，你也会使用它们，与它们交互，并受到它们的指导。了解它们背后的过程很重要，这有助于你从它们中获得最佳效果。

第二次修订是在第一次修订的基础上进行的。许多章节和练习是相似的，尽管引入了新概念，并且练习的范围也扩大了。那些已经阅读过第一次修订的人应该能够快速地通过这本书，并在途中获取新的知识，并参与额外提出的活动。对于那些是这本书的新读者，鼓励他们花时间，做练习，并进行实验。如有任何疑问，请随意破解代码以理解它，并随时提出。

由于这是一本面向程序员的书籍，我们假设你有一些编程知识和 Python 本身的知识。因此，对于 Python 代码本身在做什么的解释很少，除非它是模糊的。

# 本书涵盖的内容

第一章，*数据挖掘入门*，介绍了我们将要使用的技术，以及实现两个基本算法以开始。

第二章，*使用 scikit-learn 进行分类*，涵盖了分类，这是数据挖掘的关键形式。你还将了解一些使你的数据挖掘实验更容易执行的结构。

第三章，*使用决策树预测体育比赛赢家*，介绍了两种新的算法，决策树和随机森林，并使用它通过创建有用的特征来预测体育比赛的赢家。

第四章，*使用亲和分析推荐电影*，探讨了基于以往经验推荐产品的问题，并介绍了 Apriori 算法。

第五章，*特征和 scikit-learn 转换器*，介绍了可以创建的更多类型的特征，以及如何处理不同的数据集。

第六章，*使用朴素贝叶斯进行社交媒体洞察*，使用朴素贝叶斯算法自动解析来自社交媒体网站 Twitter 的基于文本的信息。

第七章，*使用图挖掘进行推荐跟踪*，将聚类分析和网络分析应用于找到在社交媒体上值得关注的良好人选。

第八章，*使用神经网络战胜 CAPTCHA*，探讨了从图像中提取信息，然后训练神经网络以在那些图像中找到单词和字母。

第九章，*作者归属分析*，探讨了通过提取基于文本的特征和使用支持向量机来确定给定文档的作者是谁。

第十章，*聚类新闻文章*，使用 k-means 聚类算法根据内容将新闻文章分组。

第十一章，*使用深度神经网络在图像中进行目标检测*，通过应用深度神经网络确定图像中显示的对象类型。

第十二章，*处理大数据*，探讨了将算法应用于大数据的工作流程以及如何从中获得洞察。

附录，*下一步*，逐章讲解，为深入理解所介绍的概念提供下一步的指导。

# 你需要这本书的什么

你需要一台计算机或访问一台计算机来完成这本书。计算机应该是相当现代的，但不需要过于强大。任何从大约 2010 年开始的现代处理器和 4GB 的 RAM 就足够了，你很可能在较慢的系统上也能运行几乎所有代码。

这里有一个例外，即在最后两章中。在这些章节中，我逐步介绍了使用亚马逊的云服务（AWS）来运行代码。这可能会花费你一些钱，但优点是比在本地运行代码所需的系统设置要少。如果你不想为这些服务付费，所使用的工具都可以在本地计算机上设置，但你肯定需要一个现代系统来运行它。至少需要 2012 年制造的处理器和超过 4GB 的 RAM。

我推荐使用 Ubuntu 操作系统，但代码应该能在 Windows、Mac 或任何其他 Linux 变体上良好运行。不过，你可能需要查阅系统文档来安装一些东西。

在这本书中，我使用 pip 来安装代码，这是一个用于安装 Python 库的命令行工具。另一个选择是使用 Anaconda，可以在以下网址找到：[`continuum.io/downloads`](http://continuum.io/downloads)

我也使用 Python 3 测试了所有代码。大多数代码示例在 Python 2 上无需修改即可运行。如果你遇到任何问题，并且无法解决，请发送电子邮件，我们可以提供解决方案。
