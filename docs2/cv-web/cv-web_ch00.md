# 前言

计算机视觉是计算机科学中近年来获得广泛重视的流行领域之一。此外，个人计算机的威力不断增强，从而为开发者打开了直接在终端用户机器上使用客户端脚本运行计算机视觉算法的大门。如今，网络中最流行的编程语言是 JavaScript。它允许我们开发复杂的算法并在网络浏览器中直接运行它们；这解决了几个主要问题——用户只需浏览器即可运行网络应用程序，而对于开发者来说，服务器负载更低。在这本书中，我们将提供最流行的 JavaScript 库的全面概述，并讨论它们提供的技巧，以帮助您在令人兴奋的领域，如图像处理和计算机视觉，迈出第一步。本书通过直观概述每个算法并展示库使用的清晰示例来涵盖计算机视觉方法。

# 本书涵盖的内容

第一章，*数学从未如此简单!* 涵盖了您在使用计算机视觉库和开发自己的应用程序时所需的几乎所有必要数学运算。

第二章，*将浏览器变成 Photoshop*，介绍了在计算机视觉领域常用的最流行的图像处理技术。

第三章，*人人都能轻松进行物体检测*，提供了照片和视频中物体检测的详细概述，从基本的示例，如通过颜色检测物体，到更复杂的示例，如特征检测。

第四章，*微笑挥手，你的面部已被追踪!* 讲述了面部和面部粒子的检测。此外，它还提供了一个头部跟踪的示例。

第五章，*愿 JavaScript 与你同在！用动作控制浏览器*，将物体检测的主题扩展到物体跟踪，并提供了详尽的示例。它还演示了如何使用手势或头部运动创建人机界面。

第六章，*接下来是什么？* 总结了本书中我们将要做的一切。此外，它还提供了对这里未展示的几个库的参考。

# 你需要这本书什么

本书使用 JavaScript 概述了计算机视觉领域的各种方法。需要具备初级的 JavaScript 知识。如果你对计算机视觉一无所知，那完全没问题。然而，为了理解书中的概念，你需要一些基本的数学知识。首先，你只需要一个网络浏览器和你的最爱文本编辑器。本书将引导你从计算机视觉的基础到 JavaScript 世界中最复杂的算法。

# 这本书适合谁阅读

如果你想要一种更简单的方法来在 Web 应用程序中使用计算机视觉的力量，你会发现这本书很有趣。对于那些想在网站上实现人机界面的人来说，例如创建一个游戏，这也会很有益。即使你不想在你的网站上添加计算机视觉功能，也请看看我们这里有什么——拥有计算机视觉算法的网站可能会成为未来的趋势。即使现在，你也可以通过使用客户端脚本进行计算机视觉来受益——你不需要为后端购买额外的机器。
