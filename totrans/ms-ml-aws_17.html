Section 5: Optimizing and Deploying Models through AWS

After mastering the use of various tools regarding machine learning on AWS, one important step that the readers have to master is optimizing these models and deploying them to production. In this part of the book, we discuss how machine learning models trained using AWS tools can be optimized and made ready to be deployed in production environments

This section contains the following chapters:

  • Chapter 13, Creating Clusters on AWS
  • Chapter 14, Optimizing Models in Spark and SageMaker
  • Chapter 15, Tuning Clusters for Machine Learning
  • Chapter 16, Deploying Models Built on AWS
