# 第二部分：使用 VMware 解决方案的机器学习用例

VMware 必须以完全自动化的方式处理事务，并需要能够做出决策的机器。他们目前正在尝试至少自动启动第一项补救措施，并且只有在机器没有响应或以预期的方式行为时，才会寻求帮助。这就是今天趋势所在，因为人们正在用机器学习技术进行创新。VMware 正在为基于机器学习的响应基础设施和应用做好准备，以便能够针对它们采取行动，并且不应依赖于任何子系统，通过精确地说明实时场景中对系统的影响。在接下来的几章中，我们将了解管理工具集，重点是机器学习技术。

本节包含以下章节：

+   第六章，*基于机器学习的智能日志管理*

+   第七章，*云中的机器学习作为服务*

+   第八章，*基于 Skyline 的机器学习规则引擎*

+   第九章，*使用 vRealize Code Stream 的 DevOps*

+   第十章，*使用机器学习转型 VMware IT 运营*
