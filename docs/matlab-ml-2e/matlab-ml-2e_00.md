# 前言

MATLAB 是一个综合编程环境，被许多研究人员和数学专家用于机器学习。本书将帮助你学习使用 MATLAB 进行机器学习和深度学习的基本概念，并通过高级应用来提高你的基本技能。

你将从探索 MATLAB 环境为机器学习提供的工具开始，了解如何轻松与 MATLAB 工作空间交互。然后，我们将转向数据清洗、挖掘以及分析机器学习中的各种类型数据，你将看到如何在图表上可视化数据值。接着，你将了解不同的分类和回归技术以及如何使用 MATLAB 函数将它们应用于你的数据。进一步地，你将理解神经网络的基本概念，并执行数据拟合、模式识别和聚类分析。你还将探索用于降维以提高性能的特征选择和提取技术。最后，你将学习如何利用 MATLAB 工具进行深度学习和管理卷积神经网络。

到本书结束时，你将学会如何在现实世界的案例中综合运用所学知识，涵盖主要的机器学习算法，并在使用 MATLAB 深入研究机器学习时感到自信。

# 本书面向对象

本书适合希望使用 MATLAB 进行机器学习和深度学习的机器学习工程师、数据科学家、深度学习工程师和计算机视觉/自然语言处理工程师。你应该对编程概念有基本的了解。

# 本书涵盖内容

*第一章*，*探索 MATLAB 在机器学习中的应用*，介绍了机器学习，这是基于算法和数学模型发展的人工智能的一个分支，能够“学习”数据并自主适应以根据设定的目标提高其性能。得益于这种学习能力，机器学习被广泛应用于数据分析、计算机视觉、文本翻译、语音识别、医疗诊断和金融风险评估等领域。机器学习是一个不断发展的研究领域，正在改变许多科学和工业领域的面貌。本章的目标是提供一些介绍、背景信息和 MATLAB 工具的基本知识。此外，还将介绍机器学习的基本概念。

*第二章*，*在 MATLAB 中处理数据*，探讨了如何在 MATLAB 中导入和组织我们的数据。如今，生成数据的数量是巨大的；智能手机、信用卡、电视、计算机、家用电器、传感器、家庭系统、公共交通和私人交通等只是生成数据无缝的设备的一些例子。此类数据被存储并用于各种目的。其中之一是使用机器学习算法进行数据分析。为了在 MATLAB 中导入和组织我们的数据，您应该熟悉 MATLAB 工作空间，以便尽可能简化操作。然后，我们将分析收集到的数据的可用格式以及如何将数据移动到 MATLAB 中或从中移出。我们还将探讨如何处理分组变量和分类数据的数据类型，以及如何从工作空间导出数据，包括单元数组、结构数组和表格数据，并将其保存为 MATLAB 支持的文件格式。最后，我们将了解如何以正确的格式组织数据，以便进行下一阶段的数据分析。

*第三章*，*使用分类和回归进行预测*，展示了如何使用最近邻分类一个对象，以及如何在 MATLAB 环境中执行准确的回归分析。分类算法基于我们的观察返回准确的预测。从一组预定义的类标签开始，分类器根据训练模型将每个输入数据分配一个类标签。回归将一组自变量与一个因变量相关联。通过这种技术，我们可以了解当自变量变化时，因变量的值是如何变化的。

*第四章*，*聚类分析和降维*，探讨了聚类方法，这些方法旨在在数据集中找到隐藏的模式或分组。这些算法通过选择基于元素之间相似性的聚类，无需任何标签来学习，从而识别分组。降维是将具有许多变量的数据集转换为具有较少维度但确保相似信息的过程。特征选择方法试图找到原始变量的子集。特征提取通过将其转换为新的特征来降低数据的维度。本章向我们展示了如何将数据划分为聚类，或相似项的分组。我们还将学习如何选择最能代表数据集的特征。

*第五章*, 《介绍人工神经网络建模*》深入探讨了**人工神经网络**（**ANNs**），它包括用于数据学习和分类的数据结构和算法。通过神经网络技术，程序可以通过示例学习并创建一个内部规则结构来对不同的输入进行分类。MATLAB 提供了算法、预训练模型和应用程序来创建、训练、可视化和模拟 ANNs。在本章中，我们将了解如何使用 MATLAB 构建一个基于 ANNs 的模型来预测值和分类数据。

*第六章*, 《深度学习和卷积神经网络*》考察了深度学习，这是一种基于多层 ANNs 的机器学习技术，它使许多应用达到了高度的准确性。深度神经网络能够模拟输入和输出数据之间的复杂关系。其中最成功的应用之一是计算机视觉，包括分类、图像回归和目标检测等任务。例如，深度神经网络能够生成一个分层表示的对象，其中每个对象都由一组具有视觉原语形式（如特定的边缘、方向线、纹理和重复模式）的特征来识别。卷积网络的特点是卷积层，它使用过滤器在局部区域分析数据并产生一个激活图。然后，这些激活图通过池化层进行处理，池化层将低分辨率数据聚合起来，以降低表示的维度并提高处理效率。然后，卷积层和池化层交替多次，直到图像由一个低分辨率的激活图来表示。在本章中，我们将学习深度学习的基本概念，并了解如何在 MATLAB 环境中实现基于卷积网络的算法。

*第七章*, 《使用 MATLAB 进行自然语言处理*》探讨了**自然语言处理**（**NLP**），它能够自动处理通过口语或书面语言传达的信息。这项任务充满难度和复杂性，很大程度上是由于人类语言固有的歧义性。为了使机器学习能够以人类典型的方式与世界互动，不仅需要存储数据，还要教会机器如何将其同时转换为有意义的概念。随着自然语言与环境互动，它会产生预测性知识。在本章中，我们将学习 NLP 的基本概念以及如何构建一个对句子进行标注的模型。

*第八章*, 《图像处理与计算机视觉 MATLAB》，涵盖了计算机视觉，这是一个研究如何处理、分析和理解视觉数据内容的领域。在图像内容分析中，我们使用大量的计算机视觉算法来构建我们对图像中对象的了解。计算机视觉涵盖了图像分析的各个方面，如物体识别、形状分析、姿态估计、3D 建模和视觉搜索。人类擅长识别和识别他们周围的事物！计算机视觉的目标是使用计算机准确模拟人类的视觉系统。在本章中，我们将了解计算机视觉的基本概念以及如何使用 MATLAB 实现一个用于物体识别的模型。

*第九章*, 《使用 MATLAB 进行时间序列分析与预测》, 深入探讨了时间序列数据，这基本上是一系列随时间收集的测量值。这些测量值是根据一个预定的变量和固定的时间间隔进行的。时间序列数据的一个主要特征是顺序很重要。我们收集的观测值列表是按照时间线排序的，它们出现的顺序在很大程度上揭示了潜在的规律。如果你改变顺序，这将完全改变数据的意义。顺序数据是一个广义的概念，包括任何以顺序形式出现的数据，包括时间序列数据。在本章中，我们将学习顺序数据的基本概念以及如何构建描述时间序列或任何一般序列模式的模型。

*第十章*, 《推荐系统 MATLAB 工具》，探讨了推荐引擎，这是一个可以预测用户可能感兴趣的内容的模型。当我们将其应用于电影等场景时，这变成了电影推荐引擎。我们通过预测当前用户可能会如何评分来过滤数据库中的项目。这有助于我们将用户与数据集中的正确内容联系起来。为什么这很重要？如果你有一个庞大的目录，那么用户可能找不到所有与他们相关的所有内容。通过推荐正确的内容，你可以增加消费。像 Netflix 这样的公司严重依赖推荐来保持用户的参与度。在本章中，我们将学习推荐系统的基本概念以及如何使用 MATLAB 构建电影推荐系统。

*第十一章*，*MATLAB 中的异常检测*，教您异常检测系统的基本概念以及如何在 MATLAB 中实现它。在生命周期中，一个物理系统可能会遭受故障或故障，这可能会损害其正常操作。因此，有必要引入一个能够在防止关键中断的能力范围内的异常检测系统。这被称为故障诊断系统，可以识别监控系统中可能存在的故障。寻找故障是维护干预阶段中最重要的和最具资格的阶段之一，必须以系统性和确定性的方式进行操作。为了进行完整的故障搜索，有必要分析所有可能确定它的可能原因。
