

# 第一部分：监督学习

监督学习迄今为止是商业中最常用的机器学习范式。它是实现自动化手动任务的关键。本节内容包括了可用于监督学习的不同算法，你将学习何时使用每种算法。我们还将尝试展示不同类型的数据，从表格数据到文本数据和图像数据。
