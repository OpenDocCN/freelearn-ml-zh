# 第一部分：统计学基础

本部分旨在为你提供统计和编程基础知识的了解，特别是关注多功能的 R 语言，这将为后续部分更高级的主题奠定基石。

在本部分结束时，你将牢固掌握对任何数据科学从业者理解至关重要的核心统计和编程概念。掌握这些基础技能后，你将准备好深入探索本书后续部分中等待你的更专业主题。

本部分包含以下章节：

+   *第一章*, *R 入门*

+   *第二章*, *使用 dplyr 进行数据处理*

+   *第三章*, *中级数据处理*

+   *第四章*, *使用 ggplot2 进行数据可视化*

+   *第五章*, *数据探索性分析*

+   *第六章*, *使用 R Markdown 进行有效报告*
